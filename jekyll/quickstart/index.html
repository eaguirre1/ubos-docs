Get the image

To run UBOS on the Raspberry Pi:

Download this image (approx. 400MB), uncompress, and write to a USB stick in "raw" format.

To run UBOS in VirtualBox:

Download this image (approx. 550MB), uncompress, and use as the main virtual hard drive in VirtualBox.

To boot a PC from a UBOS USB stick:

Download this image (approx. 480MB), uncompress, and write to a USB stick in "raw" format.

For a detailed series of steps, refer to the Installation section in the UBOS user documentation.

Install one or more apps

Once booted, log in from the console as root. To install ownCloud, or any other app:

# ubos-admin createsite

and answer the questions. Use owncloud as the name of the app. Details.

To show installed apps and at which virtual hosts they run:

# ubos-admin listsites

Details.

To remove all apps at a virtual host, and the site itself:

# ubos-admin undeploy --host hostname

Details.

To back up all data from your apps:

# ubos-admin backup --out all.ubos-backup

Details.

For details, refer to the UBOS user documentation.
